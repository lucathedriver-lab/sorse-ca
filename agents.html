Agent 8
Route Optimization & Dispatch
Mission

Design efficient delivery routes for the EV courier vehicle, minimizing travel time and distance while maximizing the number of deliveries completed per route.

Objectives
MINIMIZE
  • • Travel time
  • • Distance driven
  • • Charging interruptions
MAXIMIZE
  • • Deliveries per route
  • • Driver efficiency
Route Planning Rules
Group by region No backtracking Prioritize urgent Geo clustering
EV Considerations

The courier vehicle is electric. All routes must account for battery range, available charging stations, and charging time. Charging stops are added to routes when required.

Output Format

Route Name · Pickup Locations · Delivery Stops · Total Distance · Estimated Time · Priority Level · Charging Stops (if required)

🎯 Goal: Enable SORSE Courier to complete the maximum number of deliveries efficiently.
